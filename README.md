# Task6-(Login Page (HTML & CSS))

## Project Description
This project is a simple Login Page UI created using HTML and CSS without using Bootstrap.
The layout is built using Flexbox to create a clean and responsive two-section design.

## The webpage contains:
- An image section on the left side
- A login form on the right side
- Email and password input fields
- Login button
- Forgot password and register link

## Features:
- Clean and modern login interface
- Flexbox layout
- Centered container
- Responsive design structure
- Input fields with placeholder text
- Styled login button
- Forgot password link
- Register link

##  How to run it 
1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html** or open it in any web browser.
